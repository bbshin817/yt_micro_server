### `authorization/get_credential`

本APIを利用するにあたり、JWT化されたGoogle Private API Credentialを取得する

### `video/create`

video_idを新規生成し、チャンク送付用の資格情報を得る

### `video/get`

アップロード済みの動画を取得する

### `upload/chunk/video`

チャンクに分解された動画をアップロードする

### `upload/chunk/thumbnail`

サムネイルをアップロードする

### `attribute/update`

動画タイトル・説明等を更新する

### `playlist/get`

既存のプレイリストを取得する

### `playlist/create`

新たにプレイリストを作成する
